简介

Mac | Linux 开发配置手册

手册内容为「如何让一部全新的 MacBook | Linux 快速完成开发环境配置」，主要面向 Web 开发者。

  • 如果你是一名老手，本手册让你减少配置开发环境的烦恼。
  • 如果你是一名新手，那么恭喜你，你将会认识一个全新的世界。
  • 除了部分数据只支持Mac外，大部分同样适用于Linux。
  • 阅读手册
  • Github地址

gitbook安装

安装gitbook, 依赖nodejs,npm

$ npm install -g gitbook-cli

查看版本

$ gitbook -V

初始化

$ gitbook init

运行

$ gitbook serve .

生成静态页面到指定文件夹

$ gitbook build . ./mysite
